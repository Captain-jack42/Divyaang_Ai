❓ Help & Guide

🚀 Getting Started

  1. Click "Start AI" to begin object detection
  2. Allow camera access when prompted
  3. "What do you see?" - List all detected objects
  4. "How many people?" - Count specific objects
  5. "Is it safe?" - Check for potential hazards
  6. "Where is the nearest object?" - Get proximity info

🎤 Voice Commands

  • "What do you see?" - List all detected objects
  • "How many people?" - Count specific objects
  • "Is it safe?" - Check for potential hazards
  • "Where is the nearest object?" - Get proximity info

📊 Features

  • Real-time Detection: Identifies 80+ object types
  • Distance Estimation: Calculates object proximity
  • Voice Q&A: Ask questions about your environment
  • Analytics: Monitor usage statistics
  • Settings: Customize camera and audio preferences

🧰 Troubleshooting

  • Camera not starting: ensure permission is granted and no other app is using the camera.
  • Model not loading: check internet connection or ad-block/VPN settings.
  • No detections: increase sensitivity in Settings or improve lighting.
  • Audio not speaking: enable Voice Feedback and raise Volume.

⌨️ Shortcuts

  • S: Start AI
  • X: Stop AI
  • L: Toggle Listen

🔒 Privacy & Permissions

  • Video stays on your device; processing runs in the browser.
  • Analytics are session-only; no history is stored or uploaded.
  • Microphone access is used only when you click Listen.
